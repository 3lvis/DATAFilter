## Usage

`TestCheck` includes a simple method to check wheter your on testing mode or not.

```swift
import TestCheck

if Test.isRunning() {
    // Do something
}
```

## Installation

**TestCheck** is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'TestCheck'
```
